# تطبيق إدارة حسابات للمشاريع الصغيرة مع متابعة النفقات والمداخيل
==========================

## ملخص المشروع
----------------

تطبيق إدارة حسابات للمشاريع الصغيرة مع متابعة النفقات والمداخيل هو تطبيق مفتوح المصدر يهدف إلى تسهيل إدارة الحسابات والعمليات المالية للمشاريع الصغيرة. يتيح التطبيق للمشاريع إدارة حساباتها، ومتابعة النفقات والمداخيل، وتحليل البيانات المالية.

* `backend`: يحتوي على كود الخادم والواجهة الخدمية.

## إعداد البيئة باستخدام docker-compose
----------------------------------------

### خطوات الإعداد

1. تأكد من أنك تمتلك Docker وdocker-compose مثبتين على جهازك.
2. افتح ترمينال واختر الدليل الذي يحتوي على المشروع.
3. استخدم الأمر التالي لتشغيل البيئة باستخدام docker-compose:

bash
docker-compose up


### تفاصيل الإعداد

* docker-compose.yml: يحتوي على تعريفات الخادم والواجهة الخدمية والقاعدة البيانات.
* Dockerfile: يحتوي على تعريفات الخادم والواجهة الخدمية.
